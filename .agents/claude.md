# Projekt: tigube v2

## Sprache
Antworte auf Deutsch.

## Was das ist
Eine Plattform zur Vermittlung zwischen Haustierbesitzern und Tierbetreuern mit Fokus auf Suche, Profile, Kommunikation und Marktplatz.
Die App richtet sich an Owner, Caretaker und Admins.

## Wichtige Befehle
- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm run lint`

## Konventionen
- React + TypeScript mit Functional Components und Hooks.
- Styling primär über Tailwind Utilities, mobile-first.
- Routing und UI-Texte auf Deutsch (z. B. `/suche`, `/anmelden`).
- „tigube“ in UI- und Rechtstexten klein schreiben (außer rechtlich notwendige Ausnahmen).

## Memory
Lies zu Beginn jeder Session `.agents/memory/project.md` und `.agents/memory/current.md`.
Bei `update memory`: aktualisiere `current.md`, `project.md` nur bei Architektur-Änderungen.
